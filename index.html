Project 3: Review Your Experience – Pro Gamers
PHP 8 MySQL Bootstrap 5 HTML5 CSS3 PDO Scrum Git
2-person team 8 Sprints 3 User Roles ROC Mondriaan

A Scrum-driven full-stack web application where gamers can read and write product reviews. Built during Cohort 20, Period 3 (2024–2025). Users browse categories, view product details, submit reviews and like them. Administrators have a full CRUD panel with image management.

  • Dynamic category & product pages populated from MySQL
  • User registration, login & personal profile page
  • Review form with star rating & like-button per review
  • Admin panel – full CRUD + image upload/delete
  • PDO for secure database access (no SQL injection)
  • Full test plan, test-report & improvement proposal
  • Scrum artefacts: backlog, sprint reviews, retrospectives

Project Vision

Problem: No centralised place for gamers to read & write authentic product reviews. Existing platforms are too generic or lack a community-driven UI.

Solution: A responsive web-app where users browse categories, view products, read/write reviews, and like reviews. Administrators manage all content via CRUD.
