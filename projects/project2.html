Project Overview

The "Build a New Game" project was a team assignment where three students each built a unique browser game that was connected through a shared landing page — a "game island" where visitors could browse and launch any of the three games.

My responsibilities were twofold: designing and coding the main game island hub page, and independently developing a complete chess game. The chess implementation required managing the full board state, piece movement rules, check/checkmate detection, and an AI opponent — all in vanilla JavaScript.

Chess Game — Features & Mechanics

1v1 Mode
  • Two players share the same screen, taking turns.
  • Full piece movement for all 6 chess piece types.
  • Legal-move highlighting when a piece is selected.
  • Check and checkmate detection.
  • Turn indicator showing whose move it is.
1v-Computer Mode
  • AI opponent with 3 selectable difficulty levels.
  • Easy: random legal-move selection.
  • Medium: prefers captures and central control.
  • Hard: applies basic positional evaluation.
  • Visual delay on AI moves to feel natural.

Technical Challenges

Board State Management

Representing the 8×8 board as a 2D array and keeping it in sync with the visual DOM grid after every move was the core challenge — especially after captures.

Check Detection

After each move, every opponent piece had to be checked to see if its attack range reached the king — requiring an efficient piece-scan algorithm.

AI Logic

Building a tiered AI from random moves up to capture-preferring logic required clean separation of difficulty levels and a way to score moves without a full minimax tree.

What I Learned

  • Collaboration: Working in a team where each member owns a self-contained component and integrating them into a shared landing page.
  • Complex state logic: Managing board state, turn tracking, and piece rules all in one coherent data model.
  • Algorithm design: Building move generation and basic evaluation functions — a foundation for understanding game trees.
  • Responsive UI: Making the chessboard resize cleanly on different screen sizes using CSS grid and viewport units.
