Project Vision

Problem

No centralised place for gamers to read & write authentic product reviews (headsets, keyboards, games, etc.). Existing platforms are either too generic or lack a simple, community-driven UI.

Solution

A responsive web-application where users can browse product categories, view product details, read existing reviews, write new reviews, like reviews, and where administrators can manage content via full CRUD.

Key Features

  • Dynamic category & product pages populated from MySQL
  • User registration, login & personal profile page
  • Review form with rating & like-button per review
  • Admin panel – full CRUD + image upload/delete
  • PDO for secure database access (no SQL injection)
  • Full test plan, test-report & improvement proposal
  • Scrum artefacts: backlog, sprint reviews, retrospectives
  • Responsive Bootstrap 5 layout (mobile-friendly)

Evaluation

Strengths
  • Clean separation of concerns (HTML/CSS vs PHP)
  • PDO usage avoids SQL injection
  • Responsive Bootstrap layout works on mobile
  • All required artefacts delivered on time
  • 7 bugs found & fixed before demo
  • Short sprints kept the team on track
Future Improvements
