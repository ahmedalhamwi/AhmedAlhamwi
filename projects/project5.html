Project Overview

Every developer has a first portfolio. This was mine. The old version was built at the beginning of my first year at ROC Mondriaan and served as my initial digital presence — a place to present who I am and what I had worked on at the time.

While simple by today's standards, it was an important milestone. It taught me the fundamentals of structuring a real website, applying consistent styling across pages, and thinking about a visitor's experience from the outside.

Old vs. Current Version

Old Version (v1)
  • Basic static HTML & CSS — no framework
  • Fixed layout — not fully responsive
  • No dark/light mode toggle
  • Simple project list — no cards or images
  • No animations or interactive elements
  • Clear structure — Home, About, Contact sections
  • Honest representation of skills at the time
Current Version (v2 — this site)
  • Bootstrap 5 — fully responsive on all screen sizes
  • Dark / light mode with CSS variables & localStorage
  • Animated hero section with gradient theme
  • Project cards with tech badges, meta pills & hover effects
  • Dedicated project detail pages with full analysis
  • Contact form with validation
  • Bootstrap Icons throughout the UI

Technologies Used

HTML5

Structure and semantic markup for all pages.

CSS3

Styling, layout, colours and basic hover effects.

Bootstrap 5

Grid system and utility classes for the layout.

JavaScript

Basic interactivity and navigation behaviour.

What I Learned

  • Portfolio mindset: Thinking about a project from the perspective of the visitor — what do they need to see quickly to understand who I am?
  • Iterative improvement: The old version was intentionally left as a project entry in the current portfolio — a visual reminder that growth is always ongoing.
  • CSS fundamentals: Consistent colour schemes, spacing, and typography — skills that directly feed into the current version's more advanced theming.
  • Code organisation: Separating HTML structure from CSS styling from the start, establishing a habit of clean file structure.
